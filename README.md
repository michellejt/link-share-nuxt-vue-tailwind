# Link share app built with Vue, Nuxt and Tailwindcss

👉🏽 [Check it out here](https://link-share-mjt.netlify.app/)\

![screenshot](https://github.com/michellejt/link-share-nuxt-vue-tailwind/assets/88480853/eff49244-1349-439c-82ea-081cbb23d1a2)

## Setup

Make sure to install the dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev
```

## Production

Build the application for production:

```bash
# npm
npm run build

# pnpm
pnpm run build

# yarn
yarn build
```

Locally preview production build:

```bash
# npm
npm run preview

# pnpm
pnpm run preview

# yarn
yarn preview
```

Check out the [deployment documentation](https://nuxt.com/docs/getting-started/deployment) for more information.
